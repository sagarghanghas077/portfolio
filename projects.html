Global Earthquake–Tsunami Risk Prediction System

Python XGBoost Scikit-learn Matplotlib LightGBM Streamlit NumPy, Pandas

I built and deployed an end-to-end machine learning application that predicts tsunami risk using seismic data from global earthquakes recorded between 2001 and 2022. The system evaluates multiple classification models—including Random Forest, Gradient Boosting, XGBoost, and LightGBM—and automatically selects the best-performing model based on ROC-AUC. The final model is deployed as a live Streamlit web application, providing real-time risk predictions, probability scores, and feature-importance visualizations to support transparent and data-driven disaster risk assessment.
